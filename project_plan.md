# 焼肉兄弟 亀戸店 LP

## 1. プロジェクト概要
焼肉兄弟 亀戸店の1ページランディングページ。亀戸駅徒歩10分の焼肉店の魅力を伝え、来店予約・問い合わせを促進する集客用LP。ターゲットは亀戸エリア周辺の20〜50代の焼肉好き層。

## 2. ページ構成
- `/` - 1ページ完結LP（全セクションを縦スクロールで表示）

## 3. コア機能
- [x] ファーストビュー（店舗名・キャッチコピー・CTA）
- [x] お店について・こだわり
- [x] 人気メニュー4品
- [x] 基本情報（営業時間・席数・定休日）
- [x] アクセス（住所・最寄駅）
- [x] 予約・お問い合わせ（電話番号・SNS）

- [x] スクロールアニメーション
